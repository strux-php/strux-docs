# ORM Relationships

Strux ORM relationships are defined **on model properties**, not methods. They use a distinct **"Ownership"** naming theme to make database structure explicit.

:::info
Relationships are resolved dynamically via `__call()` and support lazy and eager loading.
:::

---

## Relationship Types

Strux supports:

- One-to-One (`OwnsOne`)
- One-to-Many (`OwnsMany`)
- Many-to-One (`OwnedBy`)
- Many-to-Many (`OwnedByMany`)

---

## One-to-One (OwnsOne)

```php
use Strux\Component\ORM\Attributes\OwnsOne;

#[OwnsOne(related: Profile::class, foreignKey: 'userID')]
public ?Profile $profile = null;
```

---

## One-to-Many (OwnsMany)

```php
use Strux\Component\ORM\Attributes\OwnsMany;
use Strux\Support\Collection;

#[OwnsMany(related: Comment::class, foreignKey: 'postID')]
public Collection $comments;
```

---

## Many-to-One (OwnedBy)

```php
use Strux\Component\ORM\Attributes\OwnedBy;

#[OwnedBy(related: User::class, foreignKey: 'userID')]
public ?User $author = null;
```

---

## Many-to-Many (OwnedByMany)

```php
use Strux\Component\ORM\Attributes\OwnedByMany;

#[OwnedByMany(
    related: Tag::class,
    pivotTable: 'post_tags',
    foreignPivotKey: 'post_id',
    relatedPivotKey: 'tag_id'
)]
public Collection $tags;
```

---

## Accessing Relationships

### Lazy Loading

```php
$post = Post::find(1);

foreach ($post->comments as $comment) {
    // ...
}
```

---

### Eager Loading

```php
$posts = Post::query()
    ->with('comments', 'author')
    ->get();
```

---

## Relationship Ordering

```php
use Strux\Component\Database\Attributes\OrderBy;

#[OrderBy('createdAt', 'DESC')]
public Collection $comments;
```
